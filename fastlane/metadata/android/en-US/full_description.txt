Kalinka turns your phone into the remote for a Kalinka Music Server running on a Raspberry Pi or any Linux machine: browse your library, queue music and control playback from the couch while the server plays to your hi-fi.

<b>Features</b>

• AI search and suggestions — describe what you want in plain language and the server curates matches from your library, powered by hybrid CLAP + full-text search
• Discover and browse — AI prompt suggestions, catalog cards for your sources, recent searches and favourites
• Playback and queue control — play/pause, seek, drag-to-reorder, swipe actions and batch selection
• Guided setup — a first-run wizard configures the server connection, audio output and music folders
• Server discovery — finds Kalinka servers on your network automatically via mDNS; switch between servers in the app
• Device volume — control supported output devices (e.g. Yamaha MusicCast)
• Schema-driven settings — the server settings UI is generated from the server's own schema, with simple and expert tiers

<b>Requirements</b>

Kalinka needs a Kalinka Music Server 3.0.0 or later on your network: https://github.com/madenvel/KalinkaPlayer

This is an alpha release — expect rough edges, and please report issues on GitHub.
